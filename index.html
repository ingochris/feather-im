Fast!

Start videochatting with your colleagues, friends, and family immediately in any terminal! No more time wasted on using.

Light-Weight!

ASCII powered video-compression for maximum data transmission savings.

Secure!

Encrypt your conversations on FeatherIM with the NKN decentralized blockchain network ecosystem!

Configuration

Maintain all your settings in a single config.json file. Never need to redo the setting every single time.

Multiple Colors

For better visibility, FeatherIM has a clean, beautiful color scheme that allows you to see videochat in more colors than you've ever seen at this scale.

Keybindings

You can expect common keybindings for FeatherIM. Customize bindings.json for your own liking!

    Default Keybindings

  • Quit without saving Ctrl+C
  • Save Cmd+S
  • Save and Quit Ctrl+D
  • Undo Cmd+Z

    Markdown Keybindings

  • Ctrl+A Insert Link Markdown
  • Ctrl+I Insert Image Markdown
  • Ctrl+V Insert YouTube Video
  • Ctrl+T Insert Table

Read our documentation for advanced keybindings and customization
